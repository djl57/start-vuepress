# 编写时语法规定

|符号            |作用     |  |符号    |作用|
|:--------------:|:------:|--|:------:|:-----:|
|#               |一级标题|   |-       |无序列表|
|##              |二级标题|   |1.      |有序列表|
|###             |三级标题|   |>       |引用   |
|####            |四级标题|   |! [] () |图片   |
|**文字 **        |粗体   |   |[] ()    |链接  |
|*文字 *          |斜体   |   |``（单行）``````（多行）|代码（tab缩进）|
|---             |分割线  |   |\       |转义   |
|~~文字 ~~        |删除线 |

# md语法

|符号            |作用     |  |符号    |作用|
|:--------------:|:------:|--|:------:|:-----:|
|#或==（文字下面）|一级标题|   |*或+或- |无序列表|
|##或-（文字下面）|二级标题|   |1.      |有序列表|
|###             |三级标题|   |>       |引用   |
|####            |四级标题|   |! [] () |图片   |
|**文字 **或__文字 __|粗体|   |[] ()    |链接   |
|*文字 *或_文字 _ |斜体    |  |``（单行）``````（多行）|代码（tab缩进）|
|***或---或___（三个以上即可）|分割线||\  |转义|
|~~              |删除线|
